### 关于图片链接

markdown笔记中用图路径:

` {ProgrammingBooksNotes-path}/image/{category}/{number}`

其中,

` {ProgrammingBooksNotes-path}`为Programming文件夹所在路径, 

`{category}`为书类别名, 如Linux, c++,

`{number}`为书在其类别中的编号, 如《linux高性能服务器编程》编号为1,

则路径为`../image/linux/1/{image_number}`
